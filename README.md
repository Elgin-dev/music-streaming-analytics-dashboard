## 📌 Project Overview

An interactive **data visualization dashboard** built using **Streamlit, Pandas, and Plotly** to analyze global music streaming trends and listener behavior.

This project enables users to explore insights across:
- 🌍 Countries  
- 🎧 Streaming Platforms  
- 🎼 Music Genres  
- 👤 User Behavior  

---

## 🚀 Features

- 📊 Platform market share visualization  
- 🎼 Genre popularity analysis  
- 🌍 Country-wise user distribution  
- 👤 Listening behavior insights  
- 📱 Subscription (Free vs Premium) analysis  
- 🔗 Correlation heatmaps  
- 🎯 Dynamic filters for real-time exploration  

---

## 🛠️ Tech Stack

- **Frontend/UI**: Streamlit  
- **Data Processing**: Pandas  
- **Visualization**: Plotly  
- **Language**: Python  

---

## 📂 Dataset

- **Name**: Global Music Streaming Listener Preferences  
- **Source**: Kaggle
